# Hobbies

Personal and family knowledge management for hobbies, sports, and personal development.

## Purpose

A browsable reference for practical notes, training resources, and cultural content used
by the family. Not a software project — no tests, no CI, no frameworks.

## Structure

| Directory | Content |
|-----------|---------|
| `gardening/` | Growing and picking guides (mango, watermelon, dosakai, drumstick) |
| `sports/swimming/` | Training notes, swimmer logs, Village Piranhas club info |
| `sports/soccer/` | League reference, FIFA literature |
| `sports/volleyball/` | Mission Impassibles team assets |
| `sports/cycling/` | Bike manual reference |
| `sports/tennis/` | Visual schedule resources |
| `cultural/` | Bala Vihar schedule notes |
| `autism/` | Medicaid and community resource references |
| `sdev/` | Personal development notes and references |

## Navigation

- Start with the category directory for the topic you need.
- `docs/repo-audit.md` — current state assessment (WRK-041, 2026-02-24)
- `docs/roadmap.md` — phased content and cleanup plan

## Contributing

Add markdown files directly in the relevant category directory. Keep notes practical and
concise. Do not add software scaffolding, agent frameworks, or build tooling to this repo.
